# openEar

**利用真实歌曲训练耳朵的交互式听力训练平台**

openEar 通过真实的爵士标准曲目，帮助你系统提升音程、音阶、和弦等音乐要素的听辨能力。

## 核心功能

### 🎯 练习卡片

**高度自定义的练习系统**

- **自由组合**：选择特定的音程、音阶、和弦类型进行针对性训练
- **多种类型**：音程识别、音阶练习、和弦识别、和弦进行、旋律片段
- **真实音频**：使用真实歌曲片段，而非合成音，更贴近实际音乐场景
- **灵活调整**：从基础到高级，循序渐进

### 📚 AI 秘籍

**权威分析 + 精准对应**

- **权威来源**：基于《爵士乐宝典》（The Jazz Theory Book）等权威教材的分析
- **精准对应**：每个音乐概念都关联到具体的真实歌曲片段和乐谱
- **理论实践结合**：每首歌曲配备详细的理论说明，深入理解音乐理论在实际演奏中的应用

### 📊 统计信息

**清晰的进度跟踪与效果评估**

- **实时记录**：准确率、练习时长、完成次数
- **可视化展示**：直观了解自己在各个领域的进步
- **薄弱分析**：识别需要重点加强的部分
- **数据驱动**：通过数据反馈优化练习策略

## 快速开始

1. **选择练习类型** → 选择音程、音阶、和弦等类别
2. **自定义练习内容** → 选择具体的练习项（如：纯五度、大调音阶等）
3. **开始练习** → 播放音频，识别音乐要素，获得即时反馈
4. **查看 AI 秘籍** → 学习对应歌曲的理论分析和乐谱
5. **跟踪进度** → 在统计页面查看练习数据和效果

## 特色优势
